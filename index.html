주제

친구들과 함께 여행을 계획할 때 어려움을 겪는 경우가 많습니다.
이러한 불편함을 해소하고, 더욱 즐거운 여행 준비 과정을 제공하고자 합니다.

Michiki의 목표는 다음과 같습니다!

  • 친구들과 함께 실시간으로 쉽게 여행 계획을 세울 수 있는 웹 서비스

개발 환경

항목 내용
OS Windows, Mac
Web Browser Chrome
Developer Tools VS Code, IntelliJ, Spring Tool Suite
DB Oracle DB
Server AWS ec2
Language & Framework React, Java, Spring Boot

개발 내용

  • 다국어지원: 한국어와 日本語를 지원
  • 공유 링크를 통한 친구 초대: 여행 계획을 생성한 후, 초대 링크를 통하여 쉽게 공유
  • 실시간 협업 기능: 참여자들이 동시에 여행 계획을 작성 및 수정
  • 참여자 상태 확인: 참여자들의 커서 위치와 '상태말'을 실시간으로 공유하여 누가 어떤 부분을 작업 중인지 직관적으로 파악 가능
  • 데이터 기반 여행지 추천: 서비스 사용자들의 실제 여행 계획 데이터를 분석하여, 인기 있는 여행지를 추천
