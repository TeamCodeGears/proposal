주제

친구들과 함께 여행을 계획할 때 어려움을 겪는 경우가 많습니다.
이러한 불편함을 해소하고, 더욱 즐거운 여행 준비 과정을 제공하고자 합니다.

Michiki의 목표는 다음과 같습니다!

  • 친구들과 함께 실시간으로 쉽게 여행 계획을 세울 수 있는 웹 서비스

기능 정의

  • 쉽고 직관적인 UI : 별도의 학습 없이 누구나 쉽게 사용
  • 다국어 지원 : 한국어와 일본어 지원
  • 반응형 웹: 모바일 웹 환경 지원
  • REST API : 표준화된 방식을 통해 Json 형식으로 데이터 교환

랜딩페이지

  • 소셜 로그인 : 구글 로그인
  • 사용자 인증정보 관리 : JWT 기반의 인증 관리 적용
  • 이용 약관, 개인정보 이용 동의

메인페이지

  • 개인 정보 관리: 프로필 이미지, 닉네임 등록 및 변경
  • 여행계획 : 여행계획 생성 및 삭제, 목록 조회시 필터 기능
  • 추천 여행지 : 추천 여행지 목록 제공 및 검색 기능

여행계획상세페이지

  • Google 지도 연동: 여행 계획을 작성하면서 구글 지도를 보고, 장소의 상세 정보를 바로 확인
  • 직관적인 일정 관리: 날짜별 일정 목록 조회 및 드래그 앤 드롭으로 쉽게 순서를 바꿔 효율적인 동선 작성 가능
  • 일정 및 핀 등록자 하이라이트 : 누가 어떤 일정을 등록했는지, 지도에 핀을 찍었는지 등록자를 쉽게 알아볼 수 있도록 하이라이트해서 함께 작업할 때 혼동 제거
  • 간편한 친구 초대: Hash IDs를 사용하여 민감한 데이터 노출 없이 안전한 초대링크 생성. 버튼 한번만 클릭하면 초대링크 자동복사되어, 전달만 하면 초대 완료
  • 실시간 채팅 기능: WebSocket 활용하여 계획 접속자들과 실시간 대화하며 의견 공유
  • 실시간 협업 기능: WebSocket 활용하여 각 접속자의 마우스커서 위치를 실시간 공유
  • 개인화된 여행지 장소 추천: 서비스 사용자들의 실제 여행 계획 데이터를 분석하여, 인기 있는 여행 장소를 추천 목록을 제공
